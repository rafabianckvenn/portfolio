# rafabianck.com

Personal portfolio — UI/UX & game designer, vibe coder at [Venn Studios](https://venn-studios.com/).

## Stack

- HTML / CSS / vanilla JS
- Design system: [Aurora](https://www.figma.com/design/u0gVBF2iTPYp0mXUL6h4F3/Aurora-UI)
- Hosted on GitHub Pages

## Features

- Terminal-inspired loading animation (Rafaella → Rafa)
- Aurora background ribbons + floating particles
- Responsive with mobile hamburger menu
- WCAG AA contrast compliance

## Local dev

Open `index.html` in a browser. No build step.
